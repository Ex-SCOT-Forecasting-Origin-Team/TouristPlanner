## Prerequisite: create a file for environment variable
create a file called init.sh
```
#!/bin/bash

export REACT_APP_GOOGLE_MAPS_API_KEY=<YourKey>
```
Run 
```
. init.sh
```

## Install the dependency and test
```
npm install
npm start
```
